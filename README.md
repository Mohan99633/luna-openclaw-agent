# Luna OpenClaw Agent

Luna is a cloud-hosted autonomous AI agent system built using OpenClaw and OpenRouter.

## Features

- OpenClaw AI agent framework
- OpenRouter AI model integration
- Internet search support
- Session memory and automation hooks
- GitHub integration
- Lightweight cloud deployment

## Tech Stack

- OpenClaw
- OpenRouter
- Ubuntu Linux
- Google Cloud Platform
- Node.js
- GitHub

## Project Goals

- Build scalable AI agents
- Learn AI infrastructure engineering
- Explore autonomous workflows
- Create automation-ready AI systems

## Security

Sensitive information has been removed from this repository.

Hidden data includes:

- API keys
- Tokens
- VM public IPs
- SSH configurations
- Internal authentication values
