# Luna AI Agent – Project Documentation

## Overview

Luna is a cloud-hosted AI agent built using OpenClaw and OpenRouter. The project focuses on autonomous AI workflows, intelligent agent interactions, tool integrations, and persistent cloud deployment.

The system is designed to run continuously on a Google Cloud virtual machine while supporting modular skills, automation hooks, search capabilities, and external AI models.

# Deployment Workflow

## Step 1 – Create Cloud VM

A Google Cloud VM instance was configured for hosting the AI agent.

## Step 2 – Configure SSH Access

Remote access was established using SSH and VS Code Remote SSH.

## Step 3 – Install OpenClaw

OpenClaw was installed and configured inside the VM environment.

## Step 4 – Configure OpenRouter

API integration was completed using OpenRouter models.

## Step 5 – Enable Skills

Additional tools and automation skills were installed.

## Step 6 – Run Agent

The agent was launched through OpenClaw TUI and background services.
